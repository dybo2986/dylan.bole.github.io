# Github Repo

GitHub-ready export of Dylan Bole's portfolio website.

## Included

- `index.html`
- `about.html`
- `project-*.html`
- `styles.css`
- `assets/images/`
- `assets/docs/`
- `assets/video/`
- `resume/resume.pdf`

## Notes

- This folder contains only the files used by the live website.
- Large PNG images were resized and converted to JPEG for faster loading.
- The footer email is set to `dybo2986@colorado.edu`.

## Publish

1. Create a new GitHub repository.
2. Upload the contents of this folder.
3. If using GitHub Pages, set the publishing source to the repository root.
